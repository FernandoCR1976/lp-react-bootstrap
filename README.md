# Este es un proyecto de trabajo en clase de la libreria React- Boostrap

##Se debe de instalar la siguiente dependencia ##
## npm install react-bootstrap bootstrap
Proyecto: Landing Page de una empresa de automóviles (1 hora 30 minutos)
Objetivo: Construir una landing page funcional y atractiva para una empresa de automóviles utilizando Bootstrap React.

Enfoque:

En este proyecto, vamos a aplicar los conocimientos adquiridos sobre Bootstrap React para crear una landing page que presente una empresa ficticia de automóviles.  Nos enfocaremos en utilizar componentes como Container, Row, Col, Button, Card, Carousel, Form y Modal para estructurar el contenido y lograr una interfaz de usuario agradable y responsiva.

Planificación: (15 min)

Definir la estructura:
Dividir la landing page en secciones clave:
Hero: Con una imagen llamativa, un título que capture la atención y un botón de llamado a la acción ("Ver modelos").
Características: Resaltar las ventajas de los vehículos (seguridad, tecnología, diseño) utilizando Cards.
Modelos: Mostrar un carrusel de imágenes de los diferentes modelos, con la opción de ver más detalles (implementaremos esto con un Modal).
Contacto: Incluir un formulario para que los usuarios puedan contactar a la empresa.
Diseño del Layout:
Planificar la distribución de los elementos en la página utilizando el sistema de grillas de Bootstrap (Container, Row, Col).
Considerar la responsividad para que la página se vea bien en diferentes dispositivos.
Selección de componentes:
Identificar los componentes de Bootstrap React más adecuados para cada sección:
Container para estructurar el contenido.
Card para presentar las características.
Carousel para mostrar los modelos de automóviles.
Modal para mostrar más detalles de cada modelo al hacer clic.
Form para el formulario de contacto.


Implementación:
Traducir la planificación a código, utilizando los componentes de Bootstrap React.
Prestar atención a la organización del código y la claridad.
Imágenes y contenido:
Incluir imágenes relevantes para la empresa de automóviles (puedes utilizar imágenes de ejemplo por ahora).
Generar contenido descriptivo para cada sección, incluyendo los textos de las Cards, la información de los modelos en el carrusel y las etiquetas del formulario.
Estilos:
Aplicar estilos CSS para personalizar la apariencia de la landing page, utilizando las clases de Bootstrap y estilos adicionales si es necesario.


Demostración:
Mostrar la landing page finalizada en un navegador web.
Verificar que la página sea responsiva y se visualice correctamente en diferentes dispositivos.
Análisis:
Discutir las decisiones de diseño y la implementación del código.
Explicar cómo se utilizaron los componentes de Bootstrap React para lograr el resultado final.
Preguntas:
Responder las preguntas de los estudiantes y aclarar dudas sobre el proyecto.

Recomendaciones:

Consultar la documentación oficial de Bootstrap React para obtener información detallada sobre los componentes y sus propiedades.
Explorar ejemplos de landing pages en la galería de Bootstrap para inspirarse.
¡Ser creativo! Experimentar con diferentes diseños y componentes para crear una landing page única y atractiva.
